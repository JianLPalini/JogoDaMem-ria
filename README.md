# 🧙 Jogo da Memória - Senhor dos Anéis

**Criado por Jian & Timoteo**

---

## 🧩 Descrição do Jogo

Este é um jogo de memória simples criado com HTML, CSS e JavaScript.

🎯 **Objetivo:** Encontrar todos os pares de cartas iguais no menor tempo possível.

🖱️ **Como jogar:** Clique em uma carta para virá-la. Se encontrar duas cartas iguais, elas permanecem viradas.

🏆 **Pontuação:** A cada par encontrado, você ganha +1 ponto. Se errar, perde 1 ponto.

🔄 **Jogadas:** O jogo conta quantas tentativas você fez.

⏱️ **Temporizador:** Um cronômetro mostra quanto tempo você está jogando.

---

## 🧩 Funcionalidades

- ✅ **20 cartas** (10 pares de personagens)
- ✅ **Embaralhamento** das cartas a cada partida
- ✅ **Animação 3D** ao virar as cartas
- ✅ **Placar** com pontos, jogadas e tempo
- ✅ **Botão reiniciar** para começar de novo
- ✅ **Tema Senhor dos Anéis** (cores, bordas, imagens)

---

## 🧩 Tecnologias Utilizadas

- **HTML** – estrutura da página
- **CSS** – estilos, animações e layout com Flexbox
- **JavaScript** – toda a lógica do jogo
---

## 🧩 Como Executar o Projeto

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no seu navegador
3. Vai abrir o jogo, e já pode iniciar a partida.

> Não precisa instalar nada.

## 🧩 Explicação do Código

### O que cada função faz no JavaScript:

| Função | O que faz |
|--------|-----------|
| `embaralhar()` | Mistura as cartas aleatoriamente |
| `criarCartas()` | Cria as cartas na tela |
| `clicarCarta()` | Controla o clique e vira a carta |
| `compararCartas()` | Verifica se acertou ou errou |
| `limparCartas()` | Reseta a seleção das cartas |
| `iniciarTimer()` | Começa o cronômetro |
| `fimDeJogo()` | Mostra a mensagem de vitória |
| `reiniciarJogo()` | Reseta tudo e começa de novo |
